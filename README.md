## 소개
제가 하고자 하는 것은 텍스트의 맥락을 읽어 사람의 감정을 설계하는 콘텐츠·서비스 기획자입니다. 국어국문학을 전공하며, 특히 화용론적 관점에서 동일한 문장도 상황과 맥락에 따라 전혀 다르게 해석된다는 점에 주목해왔습니다. 예를 들어 ‘괜찮아요’라는 표현은 수용이 아닌 거절로 기능하기도 합니다. 이처럼 언어는 표면적 의미를 넘어 화자의 의도와 청자의 해석이 교차하며 의미를 형성합니다.

이러한 인식은 1년간의 아르바이트 경험을 통해 더욱 구체화되었습니다. 바쁜 시간대, 불만을 가진 고객에게 해결책을 먼저 제시했지만 상황은 쉽게 풀리지 않았습니다. 그러나 “많이 불편하셨죠”라는 한 문장을 먼저 건넨 순간, 고객의 표정이 풀리며 대화의 흐름이 달라지는 것을 경험했습니다. 그때 저는 말 한마디가 막혀 있던 흐름을 트는 ‘보이지 않는 손잡이’처럼 작동한다는 것을 깨달았습니다.

이후 말투와 타이밍을 달리하며 사람의 반응을 관찰했고, 어떤 표현이 감정을 완화하고 관계를 부드럽게 이어주는지 고민하게 되었습니다. 저는 기획이란 단순히 결과를 만드는 일이 아니라, 사람의 감정이 자연스럽게 이어지도록 흐름을 설계하는 일이라고 생각합니다. 앞으로 저는 전공 지식과 실무 경험에서 얻은 감각을 바탕으로, 누구나 편안하게 받아들일 수 있는 사용자 경험을 설계하는 기획자로 성장하고자 합니다.
